# 考研英语一 · 精读标注工具

移动端可交互的 HTML 页面，用于精读考研英语一阅读文章。通过手势快速标注生词、短语和难句，导出结构化数据配合外部 AI 完成分析。

## 快速开始

用浏览器打开 `index.html` 即可使用，无需安装任何软件。

## 操作指南

### 文本标注

| 操作 | 标注为 |
|------|--------|
| **单击**单词 | 📕 生词 |
| **拖拽**选中多个词 | 📘 短语 |
| **双击**单词 | 📗 难句（自动选中整句） |

标注后自动保存到浏览器，误标可点击底部"撤销"按钮。

### 标注类型

| 类型 | 颜色 | 视觉样式 |
|------|------|---------|
| 📕 生词 | 珊瑚红 | 斜纹底纹 + 波浪下划线 |
| 📘 短语 | 青色 | 点阵底纹 + 虚线下划线 |
| 📗 难句 | 琥珀色 | 竖纹底纹 + 左右边框 |

同一单词可叠加多种类型，顶部 2px 分段色条区分。

### 标注管理

- **查看详情**：点击已标注的单词
- **删除**：在详情面板中操作，或在列表面板批量管理
- **📋 列表**：底部状态栏"列表"按钮，查看/定位/删除所有标注
- **撤销**：撤销最近一条标注
- **跳转**：点击底部统计数字，滚动到对应类型位置

### 滚动与选择

手指放在文字上**纵向滑动** → 正常滚动页面；**横向拖拽** → 选择文本。无需按住页边空白。

### 文章切换

顶部下拉框按年份分组，内置 2005-2014 全部 40 篇考研英语一阅读真题。

### 数据导出 / 导入

点击顶部"导出"按钮：

- **JSON 格式**：结构化数据（含文章全文 + 标注列表），供外部 AI 分析
- **Markdown 格式**：人可读笔记

点击"导入"可恢复之前导出的标注数据。

## 配合 AI 工具使用

1. 在工具中标注生词、短语、难句
2. 导出 JSON
3. 打开 `prompt-templates.md`，复制全部提示词
4. 将导出的 JSON 粘贴到提示词末尾
5. 发送给 Claude / ChatGPT / DeepSeek 等外部 AI

AI 会依次完成：生词释义 → 短语分析 → 语法教学（自造例句，不碰原文）→ 复习笔记。

**核心原则**：AI 教语法时禁止翻译原文，必须自造新句。学生学完后独立回到原文分析，避免假性掌握。

## 技术说明

- **零依赖**：单文件 HTML，CSS/JS 内嵌
- **离线可用**：数据存储在浏览器 localStorage
- **移动端优先**：触屏手势优化，兼容桌面端
- **隐私安全**：数据完全在本地，不上传任何服务器
