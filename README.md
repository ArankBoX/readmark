# 考研英语一 · 精读标注工具

一个移动端可交互的 HTML 页面，用于精读考研英语一阅读文章。支持触摸选择文本、可视化管理多种标注类型、导出结构化数据供外部 AI 工具分析。

## 快速开始

1. 用浏览器打开 `index.html`
2. 选择一篇文章开始阅读
3. **单击**单词选中，**拖拽**选中多个词
4. 选择文本后弹出分类面板，选择标注类型
5. 标注后的内容会高亮显示，数据自动保存到浏览器
6. 点击"导出"下载标注数据

## 操作指南

### 文本选择

| 操作 | 设备 | 效果 |
|------|------|------|
| 单击单词 | 手机/电脑 | 选中单个词 |
| 手指拖拽 | 手机 | 选中连续多个词 |
| 鼠标拖拽 | 电脑 | 选中连续多个词 |

### 标注类型

| 类型 | 颜色 | 视觉样式 | 用途 |
|------|------|---------|------|
| 📕 生词 | 珊瑚红 | 波浪下划线 | 不认识或不熟悉的单词 |
| 📘 短语 | 青色 | 虚线+下划线 | 不熟悉的词组/搭配 |
| 📗 难句 | 琥珀色 | 边框标记 | 读不通/看不懂的句子 |

### 标注管理

- **查看**：点击已标注的单词，弹出详情面板
- **删除**：在详情面板中点击"删除标注"
- **撤销**：点击底部状态栏的"撤销"按钮，撤销最近一条标注
- **跳转**：点击底部状态栏的数字，快速跳转到对应类型的标注位置

### 数据导出

点击顶部"导出"按钮，选择格式：

- **JSON 格式**：结构化数据，包含文章原文和标注列表，可直接粘贴给外部 AI 工具分析
- **Markdown 格式**：人可读的笔记格式，方便复习查看

### 数据导入

点击顶部"导入"按钮，选择之前导出的 JSON 文件，可恢复标注数据。

## 配合 AI 工具使用

1. 在精读标注工具中完成文章标注
2. 导出 JSON 格式的数据
3. 打开 `prompt-templates.md`，选择合适的提示词模板
4. 将提示词 + JSON 数据粘贴到 AI 工具中进行分析

提供四套提示词：
- **提示词一**：生词分析（词性、释义、考研搭配）
- **提示词二**：短语分析（释义、语法功能、考研要点）
- **提示词三**：难句分析（结构拆解、语法点、命题视角）
- **提示词四**：综合复习笔记（归类整理 + 复习建议）

## 技术说明

- **零依赖**：单文件 HTML，无需安装任何软件或依赖
- **离线可用**：所有数据存储在浏览器 localStorage 中
- **移动端优先**：针对触屏交互优化，同时兼容桌面端鼠标操作
- **隐私安全**：数据完全存储在本地浏览器，不上传到任何服务器
