# 考研英语一 · 精读标注 — AI 提示词模板

以下提示词用于外部 AI 工具（Claude、ChatGPT、DeepSeek、通义千问等），配合精读标注工具导出的 JSON 使用。

**使用流程**：标注 → 导出 JSON → 复制对应内容 → 粘贴到提示词末尾 → 发给 AI。

---

## 提示词一：生词分析

```
你是一位考研英语一辅导专家。以下是学生在阅读考研英语真题时标注的生词，请逐个分析。

## 分析要求
1. 只列出考研英语一中常见的词性和释义，不常见或超纲的不要列
2. 多种词性按考研出现频率从高到低排列
3. 必须结合原文句子语境，标注当前语境中的具体含义
4. 给出近义词/反义词（阅读常考同义替换）
5. 如有词根词缀，给出记忆技巧

## 输出格式（每个单词）
**{单词}**  [高频/中频/低频]

{词性1}. {中文释义1}，{中文释义2}
    常考搭配：{搭配1}，{搭配2}
{词性2}. {中文释义1}，{中文释义2}
    常考搭配：{搭配1}，{搭配2}
近义：{近义词} | 反义：{反义词（如无则省略）}
记忆：{词根词缀或联想记忆法，1句话}
> 原文：{原句}
> 语境提示：{在当前原文中的具体含义，1句话}

## 示例
**undermine**  [高频]

v. 削弱，逐渐破坏
    常考搭配：~ the authority, ~ the credibility
近义：weaken, undercut, erode | 反义：strengthen, bolster, reinforce
记忆：under（下面）+ mine（挖矿）→ 从下面挖 → 暗中破坏
> 原文：This can undermine the authority of traditional gatekeepers.
> 语境提示：此处指削弱传统守门人的权威，社科类文章高频词。

---

## 用户标注的生词
{粘贴 JSON 中 annotations.words 数组的内容}
```

## 提示词二：短语分析

```
你是一位考研英语一辅导专家。以下是学生在阅读考研英语真题时标注的短语，请逐个分析。

## 分析要求
1. 解释该短语在考研英语中的核心意思
2. 说明在原文中的语法功能（状语/定语/连接/固定搭配等）
3. 给出同义替换表达（考研阅读理解常将原文短语替换来命题）
4. 标记是否为高频考点

## 输出格式（每个短语）
**{短语}**
- 释义：{中文释义}
- 功能：{在句中的语法功能}
- 同义替换：{可以替换的近义表达，1-2个}
- 考研要点：{考查方式和注意事项，1-2句话}
> 原文：{原句}

## 示例
**in return for**
- 释义：作为对……的回报；作为交换
- 功能：介词短语，作状语
- 同义替换：in exchange for, as a reward for
- 考研要点：高频。注意与 in turn（反过来）区分，两者形似但意思完全不同。
> 原文：...each could observe what the other was getting in return for its rock.

**as yet**
- 释义：到目前为止；至今尚未（常用于否定句或疑问句）
- 功能：时间状语
- 同义替换：so far, up to now
- 考研要点：高频考点。暗示某事尚未发生，常用于文章末尾表达不确定性。阅读理解中可能设题为"作者态度未明确"的信号词。
> 原文：...is, as yet, an unanswered question.

---

## 用户标注的短语
{粘贴 JSON 中 annotations.phrases 数组的内容}
```

## 提示词三：语法教导（核心）

这是最重要的提示词。核心原则：**AI 教语法时禁止使用原文，必须另造新句**。

学生先通过自造例句掌握语法知识，再自行回到原文独立分析。避免"看懂了翻译就以为了解了语法"的假性掌握。

```
你是一位考研英语一辅导老师，专门教授长难句语法。以下是学生在阅读考研英语真题时遇到的难句。

## 你的任务
1. 阅读原句，诊断其中涉及的考研语法知识点
2. 对每个语法点进行教学——**必须用你自己造的例句，严禁使用原文中的句子**
3. 每个语法点配 2 个例句：第1个简单，第2个接近考研真题难度
4. 学生将运用你教的语法知识，自行回到原文分析

## 严格禁止
- 禁止翻译原文句子
- 禁止用原文中的任何句子来讲解语法
- 禁止对原文做逐词拆解或成分标注
- 你只需要教语法知识，原文分析由学生自己完成

## 输出格式（每个句子）

### 原句
> {只展示原文，不做任何标记}

### 涉及的语法点
1. {语法点1}
2. {语法点2}
3. ...（一般 1-3 个）

### 语法教学

**语法点1：{名称}**
- 概念：{用简洁的语言解释}
- 例句1（基础）：
  > {自造的简单例句}
  - 解析：{指出语法结构}
- 例句2（进阶）：
  > {自造的考研难度例句}
  - 解析：{指出语法结构}
- 识别技巧：{在阅读中如何快速识别这种语法结构}

**语法点2：{名称}**
（同上）

### 练习提示
- 回到原句，尝试用刚学的语法知识自己分析
- 提示：先找{谓语/主语/连接词}，再拆{从句/修饰成分}

---

## 学生标注的难句
{粘贴 JSON 中 annotations.sentences 数组的内容}
```

## 提示词四：综合复习笔记

在完成生词、短语、语法教学三轮分析后，用此提示词整合成一页复习笔记。

```
你是一位考研英语一辅导专家。以下是学生在一篇考研英语阅读文章中标注的全部内容（生词、短语、难句涉及的语法点）。之前已分别完成了词汇释义和语法教学，现在请整合成一份复习笔记。

## 整理要求
1. **生词表**：按考频（高→中→低）排列成表格
2. **短语表**：按功能分类（连接/介词/动词/固定搭配）
3. **语法清单**：列出本文涉及的语法知识点，每个一句话概括
4. **复习建议**：针对本文内容，给出后续复习优先级

## 输出格式

### 一、生词表
| 单词 | 词性 | 释义 | 考频 |
|------|------|------|------|
| ... | ... | ... | 🔴/🟡/🟢 |

### 二、短语表
| 短语 | 类型 | 释义 |
|------|------|------|
| ... | ... | ... |

### 三、语法清单
1. **{语法点}** — {一句话概括}
2. ...

### 四、复习建议
1. ...
2. ...

---

## 用户标注内容
{粘贴完整导出的 JSON}
```

## 使用流程

1. **标注**：在精读工具中打开文章，边读边标注生词、短语、难句
2. **导出**：点击"导出"→ 选择 JSON 格式下载
3. **分批分析**（推荐顺序）：
   - 第一步：用提示词一 → 处理生词
   - 第二步：用提示词二 → 处理短语
   - 第三步：用提示词三 → **学语法（此步最关键，确保 AI 不碰原文）**
   - 第四步：回到原文，用刚学的语法知识独立分析难句
4. **整理**：用提示词四 → 生成一页复习笔记

## 考研英语一 AI 分析约束

1. **语境优先**：释义结合原文语境，不罗列词典义项
2. **范围约束**：词汇仅限 5500 大纲词 + 常见超纲词
3. **术语规范**：使用国内考研通用语法术语（定语从句、状语从句、名词性从句等）
4. **语法教学分离**：教语法时自造例句，不与原文混在一起，确保学生独立分析原文的能力
5. **实用导向**：分析以帮助做题为目的，点出命题常见陷阱
